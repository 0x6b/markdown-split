# markdown-split

Split a markdown text into sections based on headings.

## Usage

```toml
[dependencies]
markdown-split = { git = "https://github.com/0x6b/markdown-split", tag = "vx.x.x" }
```

See [`examples/basic.rs`](examples/basic.rs) for usage.
